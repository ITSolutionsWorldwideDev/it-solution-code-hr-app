# IT Solutions Design Sandbox

Separate frontend-only sandbox for testing UI directions without changing the main app.

## Start

```powershell
cd "C:\Coding Projects\IT Solution Code\design-sandbox"
npm install
npm run dev
```

The sandbox runs on:

`http://localhost:3001`

## Routes

- `/` - shell overview
- `/vacancies` - vacancies list prototype
- `/vacancies/1` - vacancy detail prototype
- `/apply/1` - public apply prototype

## Notes

- Uses static mock data only
- Does not call the real backend
- Safe place to test layout, branding, cards, forms, and typography
